# 塔罗九章

## 下载

### Docker

```
docker pull apachecn0/taluo-jiuzhang
docker run -tid -p <port>:80 apachecn0/taluo-jiuzhang
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install taluo-jiuzhang
taluo-jiuzhang <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g taluo-jiuzhang
taluo-jiuzhang <port>
# 访问 http://localhost:{port} 查看文档
```
