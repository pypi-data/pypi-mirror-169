# 能量清理

## 下载

### Docker

```
docker pull apachecn0/nengliang-qingli
docker run -tid -p <port>:80 apachecn0/nengliang-qingli
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install nengliang-qingli
nengliang-qingli <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g nengliang-qingli
nengliang-qingli <port>
# 访问 http://localhost:{port} 查看文档
```
